# 更新日志

## 0.2.4

- 新增 QQ 长回复逐条普通消息发送模式，可与合并转发模式切换测试。
- 支持配置每条普通消息的最大字符数和发送间隔。
- 长回复发送钩子改为在其他结果装饰器之后执行，确保附加信息一起参与分片。

## 0.2.3

- 新增非流式 QQ 长回复合并转发分片。
- 支持配置合并转发触发字数和每个 Node 的最大字符数。
- 分片优先使用段落、换行和句末边界，并保证正文字符不丢失。

## 0.2.2

- 前端直接从绑定记录生成调试会话选项，不再依赖会话目录响应。
- 更换页面缓存版本，确保 Dashboard 加载最新脚本。

## 0.2.1

- 修复已绑定会话未出现在调试器下拉框的问题。
- AstrBot 会话目录为空或读取失败时，自动使用插件绑定记录作为兜底。
- 调试器自动选择已有绑定的会话，并显示当前有效配置。
- 页面资源增加版本参数，避免 Dashboard 缓存旧脚本。

## 0.2.0

- 新增中文首次使用向导和配置状态概览。
- 新增角色卡、提示词预设、世界书和用户设定专用编辑器。
- 新增 AstrBot Persona、会话选择和完整绑定管理。
- 新增只读扫描模拟、最终 `messages[]` 调试和真实请求查看。
- 新增文档规范化、中文校验、复制和未知导入字段合并导出。
- 数据库增加版本记录，首次升级自动创建 `0.1.0` 备份。
- 修正未绑定单选资料会隐式使用资料库第一项的问题。
- 增加角色卡 Main Prompt、PHI 覆盖策略。

## 0.1.0

- 初始版本。
- 加入角色卡、角色组、提示词预设和世界书管理。
- 加入递归扫描、生命周期、概率和聊天深度注入。
- 加入创作素材、状态栏、特殊生成模式和最终请求预览。
- 加入 AstrBot 原生 Dashboard 管理页面。
