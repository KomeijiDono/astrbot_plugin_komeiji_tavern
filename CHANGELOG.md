# 版本更新说明

## 0.5.0：长期记忆与运行仪表盘

- 新增长期记忆持久化，支持自动提取、手动新增、禁用、删除和向量检索注入。
- 新增运行仪表盘，记录真实请求的估算 Token、耗时、世界书命中、摘要状态、记忆命中和 Provider 调用量。
- 新增 `memories` 与 `runtime_metrics` SQLite 表，并加入独立保留期清理。
- 管理台新增“长期记忆”和“运行仪表盘”页面。
- 记录分支/存档设计前提：完整回溯需要插件主动保存 messages 快照，0.5.0 暂不实现完整分支树。

## 0.4.1：资料与会话备份导出

- 角色卡、提示词预设、世界书和用户设定支持分别导出 JSON。
- 支持按资料类别导出 ZIP，以及一键导出全部资料 ZIP。
- 调试器支持导出当前最终 `messages[]` 为纯 JSON。
- 新增重置前会话备份 ZIP，保存 `messages[]`、完整请求预览和插件会话状态。
- ZIP 内附清单并按资料类型分目录，导出文件名自动规避 Windows 非法字符。

## 0.4.0：自动摘要与会话生命周期

- 新增会话级滚动摘要，支持独立 Provider、触发阈值、自定义提示词、输出上限和超时。
- 历史压缩后保留近期消息，并将生成内容与静态 Summary 合并注入。
- 摘要失败自动降级且不推进进度；只读模拟不调用模型、不修改状态。
- 调试器新增摘要来源、覆盖数量、Provider、触发状态和失败原因。
- `sessions` 与 `previews` 新增更新时间索引及独立 TTL 自动清理。
- README 重构为项目主页，全部版本记录迁移至本文件。

## 0.3.10：统一请求准备与运行时常量

- 真实生成与只读模拟共用资料收集和结果序列化。
- 模拟补齐创作素材加载，与真实请求保持一致。
- 集中管理插件 ID、显示名、版本和 API 前缀。

## 0.3.9：配置页分组

- Dashboard 配置改为深色层次分组。
- 兼容旧版平铺配置并保留已有值。

## 0.3.8：限制历史消息条数

- 新增 `history_max_messages`，可只发送最近 N 条历史。
- 调试器标记由条数限制移除的旧消息。

## 0.3.7：调整上下文裁剪顺序

- 优先裁剪较旧历史并保护最近消息。
- 角色卡、Persona、摘要、记忆和 PHI 改为核心块。
- 新增历史优先裁剪和近期保留配置。

## 0.3.6：修复大型预览发送

- `/tavern preview` 使用分包转发及普通消息降级。
- 修复调试器会话 ID 重复编码。

## 0.3.5：修复本地命令误触发模型

- `/tavern preview|status|reset` 返回后不再继续调用 LLM。
- 未知子命令只显示用法。

## 0.3.4：增强 QQ 合并转发

- 长回复拆为小节点和多个转发包。
- 失败时仅降级未发送部分，避免丢失或重复。
- 新增每包节点数和发送间隔配置。

## 0.3.3：修复并发状态与配图顺序

- Web 特殊生成、重置和命令统一使用会话锁。
- 自动配图延后至正文发送完成。
- 修复调试 Persona、仓库地址及 data URL 校验问题。

## 0.3.2：性能与并发优化

- 优化中日韩 token 估算。
- 向量 embedding 增加 LRU 缓存及错误降级。
- SQLite IO 异步化并增加会话级并发锁。
- 自动配图增加并发限制。

## 0.3.1：改进会话选择

- 绑定和调试会话改为可输入搜索框。
- 补齐版本标注。

## 0.3.0：新增自动配图

- 支持在模型回复后异步生成并补发图片。
- 生图能力为软依赖，异常不会阻断文本回复。
- 增加模式、尺寸、额度及提示词配置。

## 0.2.6：支持纯文本提示词导入

- 支持 `.txt` 和 `.md` 提示词预设。
- 自动处理首尾单层引号并保留默认提示块顺序。

## 0.2.5：增强普通消息分片重试

- 增加逐片日志、失败重试和等待配置。
- 分片最终失败时停止后续发送，避免重复。

## 0.2.4：新增 QQ 普通消息分片

- 长回复可拆为多条普通 QQ 消息。
- 支持每条最大字符数和发送间隔。

## 0.2.3：新增 QQ 合并转发分片

- 支持触发字符数和 Node 最大字符数。
- 优先按段落、换行及句末安全切分。

## 0.2.2：修复调试会话来源

- 从绑定记录生成会话选项。
- 更新页面缓存版本。

## 0.2.1：完善绑定会话兜底

- 会话目录为空时使用绑定记录。
- 自动选择有效绑定并显示实际配置。

## 0.2.0：完成核心角色扮演工作流

- 增加中文向导、资料编辑器、绑定管理和调试器。
- 增加 Persona/会话目录、文档校验和未知字段往返保留。
- 修复未绑定资料隐式生效并增加角色卡覆盖策略。

## 0.1.0：初始版本

- 支持角色卡、角色组、预设、世界书和创作素材。
- 支持递归扫描、生命周期、概率、深度注入及请求预览。
- 提供 AstrBot Dashboard 管理页面。
